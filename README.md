# neighborhood-map
View and filter BART locations in the nearest neighborhood

# Install
Requires Node v6+

    $ unzip neighborhood-map.zip
    $ npm install

# Usage
    
    $ npm run build:js && npm run minify:css && npm build:views    
    $ npm run start

# Test

    $ npm test
