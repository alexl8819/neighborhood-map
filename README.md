# neighborhood-map
View and filter BART locations in the nearest neighborhood

# Install
Requires Node v6

    $ unzip neighborhood-map.zip
    $ npm install

# Usage
    
    $ npm run start

# Test
Note: CSS-Validator will fail on warnings, they are ignored

    $ npm test
